# v0.2 adapter and topology baseline

The reviewed 2026-07-13 v0.2 gate used CodeGraph 1.3.0 and Spectra index v4.

## Real-repository adapter routes

| Repository | Framework | CodeGraph routes | Spectra routes | Shared | Resolved Spectra routes |
| --- | --- | ---: | ---: | ---: | ---: |
| full-stack-fastapi | FastAPI + React | 0 | 18 | 0 | 16 |
| laravel | Laravel | 1 | 1 | 1 | 0 |
| nestjs-starter | NestJS | 1 | 1 | 1 | 1 |
| spring-petclinic | Spring MVC | 17 | 17 | 17 | 17 |
| vapor-template | Vapor | 3 | 3 | 3 | 3 |

The parity gate passed with all 22 CodeGraph route labels present in Spectra (100% reference-route recall). Spectra emitted 40 routes in total, including 18 FastAPI routes where CodeGraph 1.3.0 emitted none. Laravel's starter route uses an inline closure, so it has no static handler edge to resolve.

## Existing topology regression corpus

The pinned ripgrep, Tokio, and rust-analyzer corpus completed all nine deterministic prompts on v0.2. Median text-only CodeGraph context was 5,819 estimated tokens; median Spectra metadata was 148 estimated tokens, a 97.5% metadata reduction before vision-token accounting. Mean expected-anchor path recall was 74.1% for CodeGraph and 79.6% for Spectra. Spectra cold indexing took 198 ms, 592 ms, and 1,701 ms respectively, versus 1,386 ms, 5,127 ms, and 21,747 ms for CodeGraph.

These token figures exclude image-token charges. No model credential was available for this run, so image-inclusive provider tokens and blind answer-quality scoring remain unset; the deterministic route, anchor, latency, and payload results are the release evidence available for this checkpoint.
